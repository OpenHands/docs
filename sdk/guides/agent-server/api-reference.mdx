description: HTTP and WebSocket API for the OpenHands Agent Server
---

The OpenHands Agent Server exposes a REST and WebSocket API for managing conversations, events, files, and more. Use this API to integrate the agent server into your applications and workflows.

## Getting Started

To use the REST API, you need a running agent server. See the deployment guides:

- **[Local Agent Server](/sdk/guides/agent-server/local-server)** — Run in the same process for development
- **[Docker Sandboxed Server](/sdk/guides/agent-server/docker-sandbox)** — Isolated Docker containers for production
- **[API Sandboxed Server](/sdk/guides/agent-server/api-sandbox)** — Connect to a hosted agent server via API

## Endpoints

### Conversations

The core API for managing agent conversations — create, run, pause, and query conversations.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | [`/api/conversations`](/sdk/guides/agent-server/api-reference/conversations/start-conversation) | Start a new conversation |
| `GET` | [`/api/conversations`](/sdk/guides/agent-server/api-reference/conversations/batch-get-conversations) | List conversations |
| `GET` | [`/api/conversations/{id}`](/sdk/guides/agent-server/api-reference/conversations/get-conversation) | Get conversation details |
| `PATCH` | [`/api/conversations/{id}`](/sdk/guides/agent-server/api-reference/conversations/update-conversation) | Update a conversation |
| `DELETE` | [`/api/conversations/{id}`](/sdk/guides/agent-server/api-reference/conversations/delete-conversation) | Delete a conversation |
| `POST` | [`/api/conversations/{id}/run`](/sdk/guides/agent-server/api-reference/conversations/run-conversation) | Resume running a conversation |
| `POST` | [`/api/conversations/{id}/pause`](/sdk/guides/agent-server/api-reference/conversations/pause-conversation) | Pause a running conversation |
| `GET` | [`/api/conversations/search`](/sdk/guides/agent-server/api-reference/conversations/search-conversations) | Search conversations |
| `GET` | [`/api/conversations/count`](/sdk/guides/agent-server/api-reference/conversations/count-conversations) | Count conversations |
| `POST` | [`/api/conversations/{id}/ask_agent`](/sdk/guides/agent-server/api-reference/conversations/ask-agent) | Ask the agent a question |
| `POST` | [`/api/conversations/{id}/condense`](/sdk/guides/agent-server/api-reference/conversations/condense-conversation) | Condense conversation history |

### Events

Send messages and retrieve events from a conversation.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | [`/api/conversations/{id}/events`](/sdk/guides/agent-server/api-reference/events/send-message) | Send a message |
| `GET` | [`/api/conversations/{id}/events`](/sdk/guides/agent-server/api-reference/events/batch-get-conversation-events) | List events |
| `GET` | [`/api/conversations/{id}/events/{event_id}`](/sdk/guides/agent-server/api-reference/events/get-conversation-event) | Get a specific event |
| `GET` | [`/api/conversations/{id}/events/search`](/sdk/guides/agent-server/api-reference/events/search-conversation-events) | Search events |
| `GET` | [`/api/conversations/{id}/events/count`](/sdk/guides/agent-server/api-reference/events/count-conversation-events) | Count events |
| `POST` | [`/api/conversations/{id}/events/respond_to_confirmation`](/sdk/guides/agent-server/api-reference/events/respond-to-confirmation) | Respond to a confirmation prompt |

### Bash

Execute shell commands and manage bash sessions.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | [`/api/bash/execute_bash_command`](/sdk/guides/agent-server/api-reference/bash/execute-bash-command) | Execute a bash command (blocking) |
| `POST` | [`/api/bash/start_bash_command`](/sdk/guides/agent-server/api-reference/bash/start-bash-command) | Start a bash command (non-blocking) |
| `GET` | [`/api/bash/bash_events/`](/sdk/guides/agent-server/api-reference/bash/batch-get-bash-events) | List bash events |
| `GET` | [`/api/bash/bash_events/{event_id}`](/sdk/guides/agent-server/api-reference/bash/get-bash-event) | Get a specific bash event |
| `GET` | [`/api/bash/bash_events/search`](/sdk/guides/agent-server/api-reference/bash/search-bash-events) | Search bash events |
| `DELETE` | [`/api/bash/bash_events`](/sdk/guides/agent-server/api-reference/bash/clear-all-bash-events) | Clear all bash events |
