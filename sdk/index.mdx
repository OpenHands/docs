A clean, modular SDK for building AI agents. This repository provides two Python packages:

- `openhands.sdk` — core agent framework (agent, conversation, context, LLM, events, tools API)
- `openhands.tools` — production-ready tool implementations (Bash, File Editor, Task Tracker)

## Goals:

- **Simple mental model**: short, focused modules with clear responsibilities
- **Strongly typed**: strict type checking with Pydantic validation
- **Extensibility**: easy to add new tools, LLMs, and components
- **MCP-first tool system**: MCP-compatible input/output schemas for all tools

## Quick start

```bash
git clone https://github.com/All-Hands-AI/agent-sdk
make build

# run an example
uv run python examples/01_hello_world.py
```

## Core concepts

Comprehensive architecture documentation is available in the [`architecture/`](./architecture/) folder:

- **[Overview](/agent-sdk//architecture/overview)** - High-level component interactions and design principles
- **[Agent](/agent-sdk/architecture/agent)** - orchestrates LLM + tools to take actions Agent execution flow
- **[Tool & MCP](/agent-sdk/architecture/tool)** - Tool framework, built-ins, runtime tools, and MCP integration. Each with strongly-typed actions + observations (input_schema/output_schema). 
- **[LLM Integration](/agent-sdk/architecture/llm)** - Provider support, message types, and advanced features
- **[Conversation](/agent-sdk/architecture/conversation)** - Stateful driver for user-agent interaction
- **[Context](/agent-sdk/architecture/context)** - Microagents and knowledge that condition the agent
